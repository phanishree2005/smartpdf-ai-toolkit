# SmartPDF – AI Document Toolkit 🚀

SmartPDF is an AI-powered PDF processing platform designed for students, developers, and professionals. It provides powerful tools to manage, analyze, and interact with PDF documents.

## ✨ Features

### 📄 PDF Tools

* Merge PDFs
* Split PDFs
* Compress PDFs
* Rotate Pages
* Extract Pages
* Delete Pages
* Reorder Pages

### 🤖 AI Features

* Chat with PDF
* AI Document Summary
* Keyword Extraction
* Smart Searc
